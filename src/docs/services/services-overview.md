# Proxy Overview

dFlow by default comes with nginx proxy support. Parameters can be configured
from `proxy` tab.

### Nginx

Here're the list of parameters you can configure for nginx proxy

| Property | Default | Description |
| -------- | ------- | ----------- |
| January  | $250    |
| February | $80     |
| March    | $420    |
